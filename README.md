# 6월 단거리 인자작 & 마신표 추천기 - master.mdb 확장판

기존 UI를 유지하면서 master.mdb에서 추출한 데이터를 반영한 버전입니다.

추가/변경 사항:
- 캐릭터 적성표 탭 추가
- 캐릭터 추천 결과에 잔디/더트/거리/각질 적성 표시
- 서포트카드 힌트 스킬을 support_card_data.skill_set_id + single_mode_hint_gain 기준으로 교체
- 캐릭터 보유/각성 스킬을 available_skill_set 기준으로 교체
- 데이터 관리 버튼 초기 연결 오류 수정

배포:
GitHub Pages 또는 Netlify에 `index.html`을 덮어올리면 됩니다.
